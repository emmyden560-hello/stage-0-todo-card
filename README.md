# Todo Card Component

A responsive and accessible todo card component built with HTML and CSS.

## Features

- **Responsive Design**: Adapts to different screen sizes.
- **Accessibility**: Includes proper ARIA labels and keyboard navigation.
- **Styling**: Modern dark theme with smooth transitions.

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge).

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd Testable-todo-item-card
   ```

2. Open `index.html` in your web browser.

## Usage

### Interacting with the Card

- **Complete Todo**: Click the checkbox to mark the todo as complete.
- **Edit Todo**: Click the "Edit" button to modify the todo details.
- **Delete Todo**: Click the "Delete" button to remove the todo.

### Customization

You can customize the appearance and behavior of the card by modifying the `style.css` file.

## Testing

### Manual Testing

1. Open `index.html` in your browser.
2. Verify that the card displays correctly.
3. Test the complete, edit, and delete functionality.
4. Check responsiveness by resizing the browser window.

### Automated Testing

This project includes automated tests using Playwright.

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the tests:
   ```bash
   npx playwright test
   ```

## Technologies Used

- **HTML5**: For semantic structure.
- **CSS3**: For styling and layout.
